## How you can help

- Star the repo if you like the project.
- Open issues for ideas before sending PRs.
- Share the project with others who work in franchise operations.
- Help improve docs, testing, performance, security, or UI polish.

## Contribution flow

1. Open an issue first.
2. Wait for feedback.
3. Submit a focused PR.
