V2 added the day-to-day product layer on top of the core platform: dashboards, knowledge sharing, scheduling, admin views, and food-service utilities. Demo mode and LinkedIn OAuth also landed here.

## Included

- ✓ Dashboard with today's schedule and org stats
- ✓ Task comments, upvotes, replies, and pinning
- ✓ Tag system (color-coded task categories)
- ✓ Task card view (image-rich grid layout)
- ✓ Demo mode (isolated sandbox, no account)
- ✓ LinkedIn OAuth
- ✓ Admin panel (overview, growth charts, feedback, photos)
- ✓ User count graphs / growth reporting
- ✓ Announcements
- ✓ Conversion Tool (ingredient scaling, batch sets)
- ✓ Staff Roster (multi-week scheduling, templates)
- ✓ Item List Tool (ingredient and product catalog)
- ✓ Bot members (placeholder roster slots)

## Planned

- ○ Dedicated notification system (in-app inbox, live updates, read states)
